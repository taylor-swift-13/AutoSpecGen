# AutoSpecGen
Using large language models to generate ACSL specifications for C code



base prompt：只包含简单的指令

pro prompt：包含简单的 cot 以及 examples



进行n次生成，每次生成注释后进行 m 次迭代修复，直到得到正确的规约

总的最大调用大模型次数：生成次数 m *（1 + 修复次数 n）
